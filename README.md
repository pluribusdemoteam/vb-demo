# vb-demo

## Pre-Requisites

- Windows machine not required!
- [Docker](https://www.docker.com/)


## Getting started

```
git clone https://github.com/pluribusdemoteam/vb-demo.git
cd vb-demo
```

## Build the image

```
docker build -t hello_vb .
```

## Run the image

```
$ docker run hello_vb
Hello World!
```
